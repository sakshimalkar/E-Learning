# E-Learning Management System

A full-stack e-learning web application designed for students to enroll in courses, manage their profiles, and receive course completion certificates via email.  
The system includes secure authentication, email-based password recovery, student course management, and an admin dashboard for certificate generation and student management.

## 📌 Key Features

### 👨‍🎓 Student Module
- Student registration and login
- Email-based account verification
- Forgot password and reset using **EmailJS**
- Course enrollment and access
- Profile update and account management
- Automatic certificate delivery via email after course completion

### 🧑‍💼 Admin Module
- Secure admin dashboard
- Manage students and enrolled courses
- Certificate generation and distribution
- Course management (add, update, delete)
- Student progress monitoring
