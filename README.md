# 🎓 E-Learning Management System  

A full-stack web application designed to manage online learning, allowing students to enroll in courses, track progress, and receive certificates, while admins manage users and content efficiently.

---

## 🚀 Live Demo  

👉 https://sakshimalkar.github.io/E-Learning/

---

## 💡 Overview  

This system provides a complete **student–admin learning platform** with secure authentication, course management, and automated certificate delivery via email.

---

## 🧑‍🎓 Student Functionalities  

- Register and login securely  
- Email-based account verification  
- Forgot password & reset using EmailJS  
- Enroll in courses and access content  
- Update profile and manage account  
- Receive course completion certificate via email  

---

## 🧑‍💼 Admin Functionalities  

- Secure admin dashboard  
- Manage students and enrollments  
- Add, update, delete courses  
- Generate and send certificates  
- Monitor student progress  

---

## ⚙️ Tech Stack  

**Frontend:**  
HTML5, CSS3, JavaScript  

**Backend:**  
PHP  

**Database:**  
MySQL  

**Email Service:**  
EmailJS (for password reset & certificate delivery)  

---

## 🧱 System Flow  

```
User (Student/Admin)
        ↓
Frontend (HTML/CSS/JS)
        ↓
Backend (PHP)
        ↓
Database (MySQL)
```

---

Main tables used:

## ⚙️ Setup Instructions  

### Prerequisites  
- XAMPP / WAMP  
- MySQL  
- Git  

---

### Installation  

```bash
git clone https://github.com/sakshimalkar/E-Learning.git
cd E-Learning
```

---

### Database Setup  

```sql
CREATE DATABASE elearning_db;
```

- Import the provided SQL file into MySQL  

---

### Run Project  

- Move project folder to `htdocs`  
- Start Apache & MySQL  
- Open browser:

```
http://localhost/E-Learning
```
